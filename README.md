# Face-Space
A 2D bullet hell game with fast-paced combat and score tracking.
<br />
<br />

## How to Run
- Download the release zip file.
- Extract the zip.
- Run the FaceSpace.exe file.
<br />
## Controls
- Move: WASD
- Dash: Space
- Shoot: Left Mouse-Click
- Special Ability: Right Mouse-Click
